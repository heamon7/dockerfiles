## 使用指南：
- 导出之前 wiki 的 MongoDB 数据
- 使用 docker-compose.yml 启动 docker: `docker-compose up -d`
- 导入 MongoDB 数据
- 在线修改配置文件：访问 http://host:3001 修改配置，重点是 host,db,gitlab
- 配置 Nginx
